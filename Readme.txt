1. Masukkan input plain teks dikolom input paling atas, bisa dengan input keyboard atau menggunakan file .txt.
2. masukkan kunci dikolom kedua, jumlah karakternya minimal 12 karakter.
3. lalu dibawah teks "Pilih metode:", terdapat radio button yang awal terpilihnya adalah Vigenere, klik untuk mengganti metode yang ingin digunakan.
4. lalu ada checkbox yang bertuliskan "Enkripsi", beri centang jika ingin mengenkripsi, jika ingin dekripsi, klik untuk menghapus centangnya.
5. Ketik tombol Proses Teks untuk menampilkan teks setelah dienkripsi atau dekripsi dibagian paling bawah.
